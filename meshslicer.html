Description

In the third year of my Bachelor at the AUAS I interned at the KLM XR Center of Excellence. During this internship I made the functionality for, how I can best describe it, VR fruit ninja. To achieve this, I created an algorithm which can split a single mesh into two meshes using a single plane or multiple connected planes.

What did I learn?

  • Learned how Unity stores data on meshes.
  • Learned how to make meshes using an algorithm.
  • Learned how to use 3D debugging tools.
  • Improved my debugging skills.
  • Improved my 3D math skills.
