Invasion

When I started with my Bachelor ICT Game Development at the Amsterdam University of Applied Sciences I was thrown into the unknown and had to create a game with some classmates. We had to create a game which could be played on the on-board entertainement of an airplane. The on-board entertainment had an old nintendo style controller, which meant we had to make sure our game only needed four movement keys and four action buttons. After some thinking, designing and programming, this game was the result.

A rework of this game made in Unity3D will be coming in the future! (Currently in development)
