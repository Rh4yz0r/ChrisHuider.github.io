Invasion

When I started with HBO-ICT Game Development at the Hogeschool van Amsterdam I was thrown into the unknown and had to create a game with some classmates. We had to create a game which could be played on the on-board entertainement of an airplane. The on-board entertainment had an old nintendo style controller, which meant we had to make sure our game only needed four movement keys and four action buttons. After some thinking, designing and programming, this game was the result.

This game was created in Processing using the Java programming language. It was made in collaboration with three classmates and took about half a year (one semester) to finish.

My contribution to the project was diverse. I made most of the pixelart artwork like the levels, programmed the collisions, enemy AI and some particles.

A rework of this game made in Unity3D will be coming in the future!
