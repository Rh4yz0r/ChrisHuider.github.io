Astral Drift

During the second semester in my third year of my Bachelor in ICT Game Development I worked on project Astral Drift. Astral Drift is a mobile shoot 'em up game made in collaboration with the Mediacollege Amsterdam.

We started out with a very basic prototype made by a previous team, but our client wanted it to be totally reworked. So we basically started anew, scrapping a lot of old parts of the game, reusing what was usefull. After this we started working together with five students from the Mediacollege, who would be responsible for creating the game's graphics.

During the project, we focussed on designing and documenting every step we made. We knew another team would pick this project up after we delivered it to our client, so we made readability and documentation a very high priority.

In total the project took around three months, after which we delivered the project to our client along with documents detailing everything we'd done. In the future, a different team will pick up this project in order to add more content to the game.

PS. One of the images below shows the initial prototype for the user interface, which I created in Figma for testing purposes, seeing as the user interface is one of the most important features for a mobile game.
