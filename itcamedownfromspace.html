It Came Down From Space

The second Game Development project I worked on was: It Came Down From Space. For this project we where free to make whatever game we wanted, the only restriction was that we had to use predetermined assets. I was put into a new team which consisted of seven people. We wanted to make a game which was reminiscent of Getting Over It, so we created this game.

In this game the player slings around the level while keeping the little alien's glass ball from breaking. When the glass ball hits an object it breaks and you have to start over from the last checkpoint. This game is played using only you mouse. Using left- and right mousebutton you can make the hand grab and release and by moving the mouse the alien starts slinging.
