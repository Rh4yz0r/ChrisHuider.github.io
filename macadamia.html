Description

In the third year of my Bachelor at the AUAS I interned at the KLM XR Center of Excellence. During this internship I had to rework a node system made for creating graphs containing cockpit procedure sequencing called 'Macadamia'.

What did I learn?

  • Learned how to create custom editor windows.
  • Learned how to use Unity's UIElements system.
  • Learned about scriptable object back-end.
  • Learned about the positives and negatives of using scriptable objects.
  • Learned how to use Unity's experimental GraphView.
  • Learned how to design user interfaces for developer tools.
  • Learned the basics of user experience in relation to developer tools.
