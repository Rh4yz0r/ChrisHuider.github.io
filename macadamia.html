Macadamia

Description

Macadamia is a graph system in charge of storing data on flight procedures for the Royal Dutch Airlines' (KLM) cockpit procedure training. It was made using Unity's GraphView with data storage being done using ScriptableObjects. Madacamia is used to store the individual steps for every procedure present in the cockpit procedure training. The user can be guided through each procedure or go through them on their own and depending on which Macadamia will initiate different triggers like showing the next step in the user interface (UI), showing which button or knob to activate next, etc.

Backstory

I worked on Macadamia during my internship at the KLM. An ex-colleague made the first version of Macadamia, but after he left, the team was left without any knowledge of it's inner workings. Some adjustments had to be made to Macadamia, but because no-one knew how it worked I took on the task of finding out. Because the back-end of Macadamia was in dire need of some cleaning and an upgrade, I volunteered to rework the entire back-end. After approximately five weeks, the rework was finished and included not only the back-end, but also the UI, data saving and formatting, reusability and making the system easily adjustable.

My contribution

I rebuilt the entire back-end of Macadamia, as well as redesigning the UI with supervision of the design team, changing the way data is saved and formatted and ultimately passing everything I've learned and worked on onto the team so they won't face the same problems again. I wrote a document on every element of Macadamia, showing how it works, how to adjust, add or delete elements and explaining simple actions for using Macadamia.
